# ShopVote

Integriert die Services von ShopVote in den JTL-Shop5

### Freigegebene JTL-Shop-Versionen
- JTL-Shop 5: ab 5.0.0

### Wichtige Upgrade-Hinweise
Beim Upgrade einer Version < 1.1.0 werden die Daten in eine neue Tabellenstruktur migriert!
Wer die alten Datenstrukturen nocht benötigt sollte zuvor folgende Tabellen sichern:
- xplugin_lfs_shopvote_article_reviews
- xplugin_lfs_shopvote_article_review_items

### Änderungen
Alle Änderungen am Plugin werden im CHANGELOG.md festgehalten.
